# SkillSync v0.1.0

Initial public release of SkillSync.

## Highlights

- Native macOS app for managing AI coding agent skills from one shared library.
- Shared source root at `~/.agent/skills`.
- 25 built-in agent presets, including Claude Code, Codex, Cursor, Kiro, GitHub Copilot, Gemini, Zed, Windsurf, OpenCode, OpenClaw, Hermes, and more.
- Directory, single-file, and overlay symlink strategies.
- Skill organize and restore workflows for moving real files into the shared source root while preserving agent-specific links.
- Git sync UI for pushing and pulling the skill library with PAT authentication.
- File watching for local skill changes.
- Custom agent definitions with configurable paths and link strategies.

## Build

- Version: `0.1.0`
- Build: `1`
- Minimum macOS: `15.0`

## Notes

This release is intended for early testing. The app is self-signed/ad-hoc signed and may require using **Open Anyway** from macOS Privacy & Security on first launch.
